### 📰 4. Um titulo de issue no GitHub comprometeu 4.000 maquinas de devs

**Resumo:** "Clinejection" — atacante injetou instrucao maliciosa em titulo de issue no GitHub. Bot de triagem com IA do Cline (claude-code-action) executou o titulo como instrucao legitima, corrompeu cache do GitHub Actions, roubou token npm e publicou cline@2.3.0 — instalando OpenClaw silenciosamente em 4.000 maquinas de devs em 8h. Zero acao necessaria do usuario. Expoe vulnerabilidade estrutural em qualquer pipeline CI/CD com agente de IA.

**Fonte:** https://grith.ai/blog/clinejection-when-your-ai-tool-installs-another

**Prioridade:** 🟡 Media (alerta importante para devs — banco de pautas se nao couber hoje)

---

- **Estrutura:**
  - Slide 1 (capa): "O ataque de IA mais elegante de 2026: um titulo que corrompeu 4.000 maquinas"
  - Slide 2: "Deixa eu te explicar..." — Atacante abriu issue no Cline com instrucao maliciosa no titulo.
  - Slide 3: Bot de IA leu o titulo como instrucao legitima. Executou. Envenenou o cache do GitHub Actions. Roubou token npm.
