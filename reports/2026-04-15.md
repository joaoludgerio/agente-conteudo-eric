### 📰 PAUTA 1 — Claude Code agora faz seu trabalho enquanto você dorme

**Resumo:** A Anthropic lançou as **Claude Code Routines** em 14 de abril (research preview) — um sistema que permite criar automações recorrentes que rodam na infraestrutura da Anthropic, sem o seu computador precisar estar ligado. Você configura uma rotina uma vez: prompt, repositório, conectores — e ela executa automaticamente em horários agendados, via API ou em resposta a eventos do GitHub (push, PR aberto, issue criada). Casos de uso imediatos: triagem automática de issues, relatórios matinais, resumos de PR no Slack, monitoramento de código. Disponível para planos Pro (5 exec/dia), Max (15/dia), Team e Enterprise (25/dia).
