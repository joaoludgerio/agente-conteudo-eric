### 1. Karpathy lançou agentes que fazem pesquisa de IA enquanto você dorme

**Resumo:** Andrej Karpathy lançou o `autoresearch`: um sistema que dá uma GPU a um agente de IA e o deixa rodar experimentos de machine learning sozinho a noite toda. O agente modifica código, treina por 5 minutos, verifica se melhorou, descarta ou mantém, repete — gerando ~100 experimentos por noite sem supervisão humana. 3k estrelas no GitHub em horas. Um fork para macOS apareceu no mesmo dia. Isso é o fim do trabalho manual de tunar modelos de IA.

**Fonte original:** [github.com/karpathy/autoresearch](https://github.com/karpathy/autoresearch)

### 6. IA para o espaço filosófico: arquivos são a nova interface dos agentes

**Resumo:** Post viral no HN articula uma tese crescente: sistemas de arquivos estão se tornando a interface principal dos agentes de IA. Em vez de 100 ferramentas por API, a tendência é agente com acesso a filesystem + 5–10 ferramentas. Ecoa a filosofia Unix: quando tudo parece um arquivo, o sistema fica mais simples e composável. Claude Code e Cursor já adotam isso.

**Fonte:** [madalitso.me](https://madalitso.me/notes/why-everyone-is-talking-about-filesystems/) | HN 223 pontos

**Prioridade:** 🟢 Banco de pautas (mais técnico, ótimo para audiência de devs e tech leads)
